## Why

The Drizzle ORM schema is currently out of sync with the database structure generated by the PokeAPI extraction script. This change will update the schema to include localized names and additional tables, ensuring that the application can access the full breadth of extracted data through Drizzle.

## What Changes

- **Schema Update**: Add localized name columns (`name_en`, `name_ja`, `name_zh`) to the `pokemon` table.
- **New Tables**: Add `pokemon_forms` and `types` tables to the Drizzle schema.
- **Migration**: Generate a new SQL migration to synchronize the schema with the existing database content.

## Capabilities

### Modified Capabilities
- `vgc-data-schema`: Update to include localized names and form/type metadata.
- `multilingual-pokemon-data`: Ensure Drizzle supports the extracted localized data.

## Impact

- **Database**: The Drizzle schema will accurately reflect the structure of `vgc_pokemon.db`.
- **Types**: New TypeScript types will be available for the localized and extended data.
- **Code**: Application queries can now use the localized name fields.
