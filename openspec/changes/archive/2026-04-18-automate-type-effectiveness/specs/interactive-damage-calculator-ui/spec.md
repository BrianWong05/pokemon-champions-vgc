## MODIFIED Requirements

### Requirement: Reactive Stat Inputs
The UI SHALL update the damage results immediately upon any stat, slider, Pokémon selection, move type, or automated type effectiveness change.

#### Scenario: Visual effectiveness feedback
- **WHEN** the user selects a Pokémon or move type that results in a super effective hit
- **THEN** the effectiveness display SHALL update to show "2.0x" (or higher) and use a green color scheme.
