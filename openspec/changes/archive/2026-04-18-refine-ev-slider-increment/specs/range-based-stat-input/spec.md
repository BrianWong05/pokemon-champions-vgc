## MODIFIED Requirements

### Requirement: Range-based Stat Adjustment
The system SHALL provide a slider (range input) for each stat in the EV converter to allow for tactile value adjustments, incrementing by 4 each step.

#### Scenario: Dragging the slider
- **WHEN** the user drags the HP stat slider to a new position
- **THEN** the numeric HP EV input SHALL update to reflect the slider's value in real-time, incrementing in steps of 4.
