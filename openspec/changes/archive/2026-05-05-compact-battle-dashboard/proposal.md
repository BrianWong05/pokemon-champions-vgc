# Proposal: Compact Battle Analysis Dashboard

## Goal
Condense the `ResultsPanel` and `MoveResultColumn` UI vertically to ensure it fits on smaller laptop screens (e.g., 13" laptops) without requiring excessive scrolling, while maintaining all existing data and visual clarity.

## Strategy
1. **Vertical Space Optimization**: 
   - Reduce top-level padding and gaps.
   - Shrink the "Health Status" section height.
   - Condense the "Move Damage Assessment" list.
2. **Typography Adjustments**: 
   - Scale down massive percentage text sizes.
   - Use tighter tracking for labels.
3. **Component Refinement**:
   - Reduce card padding and border thickness where appropriate.
   - Use more horizontal space for status indicators.

## Expected Outcome
A significantly more compact dashboard that retains its "premium" aesthetic but is much more practical for smaller viewport heights.
