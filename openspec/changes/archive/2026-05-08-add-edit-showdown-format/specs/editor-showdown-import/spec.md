## ADDED Requirements

### Requirement: Editor Import Trigger
The Team Member Editor Modal SHALL provide an "Import/Edit Showdown" button.

#### Scenario: Triggering editor import
- **WHEN** the user clicks "Import/Edit Showdown" in the modal
- **THEN** the import interface SHALL appear.

### Requirement: Editor Batch Update
The Team Member Editor SHALL be updated to populate its fields when Showdown data is imported.

#### Scenario: Updating fields
- **WHEN** Showdown data is imported into the editor
- **THEN** the modal form fields SHALL update to reflect the new Pokémon configuration.
