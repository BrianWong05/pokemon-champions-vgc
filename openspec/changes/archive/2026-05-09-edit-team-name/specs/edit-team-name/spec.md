## ADDED Requirements

### Requirement: Rename existing team
The system SHALL allow users to rename an existing team from the Team Detail page.

#### Scenario: Successful team rename
- **WHEN** user clicks the edit icon next to the team name
- **AND** user types a new valid name
- **AND** user saves the changes
- **THEN** the team name is updated in the database
- **AND** the new team name is displayed on the page
