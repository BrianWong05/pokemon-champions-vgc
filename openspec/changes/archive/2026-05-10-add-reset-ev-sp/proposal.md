## Why
Users currently have to manually reset each stat field in the EV/SP Converter. A "Reset All" button would significantly improve user efficiency.

## What Changes
- Add a "Reset" button to the `EvSpConverterPage` interface.
- Implement the reset logic to clear all EV spreads to 0.

## Capabilities

### New Capabilities
- `ev-sp-reset`: Adding reset functionality to the EV/SP converter.

## Impact
- Increases convenience and speed for resetting stat spreads.
