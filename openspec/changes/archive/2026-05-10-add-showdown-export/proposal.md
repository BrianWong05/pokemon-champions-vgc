## Why
The Damage Calculator currently lacks a way to easily export the configured Pokémon set into the standard Showdown format, making it difficult for users to copy/paste their teams or sets elsewhere.

## What Changes
- Add an "Export to Showdown" functionality in the Damage Calculator interface.
- Implement a utility function to format the current Pokémon state into the standard Showdown text format.
- Add an export button to the Pokémon panel or configuration interface.
