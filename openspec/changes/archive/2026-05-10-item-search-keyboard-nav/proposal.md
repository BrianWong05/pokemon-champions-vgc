## Why
The item search dropdown currently lacks keyboard navigation, which is inconsistent with the Pokémon and move search components. Standardizing this improves usability in team building.

## What Changes
- Enhance `ItemSearchSelect` to handle keyboard events for navigation and selection.

## Capabilities

### New Capabilities
- `item-keyboard-nav-support`: Adding keyboard event listeners and navigation logic to the item search dropdown.

## Impact
- Unified keyboard-only user experience across all search components.
