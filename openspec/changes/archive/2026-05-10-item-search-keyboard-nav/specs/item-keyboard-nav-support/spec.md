## ADDED Requirements

### Requirement: Item Search Keyboard Navigation
The system SHALL support keyboard-based navigation within the item search dropdown.

#### Scenario: Verify ArrowDown navigation
- **WHEN** the item dropdown is open and the user presses ArrowDown
- **THEN** the system MUST highlight the next available item.

#### Scenario: Verify Enter selection
- **WHEN** an item is highlighted and the user presses Enter
- **THEN** the system MUST select that item.
