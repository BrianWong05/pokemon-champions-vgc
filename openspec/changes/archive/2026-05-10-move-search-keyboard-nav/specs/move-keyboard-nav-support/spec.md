## ADDED Requirements

### Requirement: Move Search Keyboard Navigation
The system SHALL support keyboard-based navigation within the move search dropdown.

#### Scenario: Verify ArrowDown navigation
- **WHEN** the move dropdown is open and the user presses ArrowDown
- **THEN** the system MUST highlight the next available move.

#### Scenario: Verify Enter selection
- **WHEN** a move is highlighted and the user presses Enter
- **THEN** the system MUST select that move.
